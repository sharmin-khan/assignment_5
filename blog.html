Question-3: What is event delegation in the DOM?


Answer : Event delegation is a technique where a single event listener is added to a parent element instead of multiple child elements. The event bubbles up from the child to the parent, allowing the parent to handle it.

Question-4: What is event bubbling in the DOM?


Answer : Event Bubbling is a behavior in the DOM where an event starts from the target element and propagates up to its parent elements.
